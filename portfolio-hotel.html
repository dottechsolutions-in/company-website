Hotel Booking Manager

ASP.NET Core • SQL Server • Bootstrap

Project Overview

This web application helps hotels manage room bookings, customer records, and payments efficiently. It provides role-based access for Admin, Receptionist, and Guests.

Key Features:
  • Room availability & booking management
  • Guest profile & payment tracking
  • Admin dashboard with analytics
  • Role-based login (Admin / Staff / Guest)
  • Invoice generation
Technologies Used:

ASP.NET Core SQL Server Bootstrap

Outcome:

The system reduced manual errors by 70% and streamlined hotel operations, improving customer satisfaction.
